# Лабораторна робота №5

Створення простого API

## Модель
Метод Гауса (5 семестр)

## Автор
Ахмадзада Вахід, група АІ-232

## Варіант
JSON POST-запит (непарний номер у списку)

curl -X POST http://127.0.0.1:5000/calculate -H "Content-Type: application/json" -d "{\"matrix\":[[2,1,-1],[-3,-1,2],[-2,1,2]],\"vector\":[8,-11,-3]}"
